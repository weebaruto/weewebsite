# Example Jobs-To-Be-Done
>Example [Jobs-To-Be-Done](../productspeak/product-jobs-to-be-done/) artifacts (and templates to follow) based on the introduction of a Foundational Observability Platform powered by OpenTelemetry.

### Purpose
Introduction to the concept of Jobs-To-Be-Done, a framework which asserts that customers don’t just “buy products” per se; they “hire” (and “fire”) solutions to do a “job” in their lives.
